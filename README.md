心形灯PCB，触摸版心形灯
==
#STC89C52RC的心形灯，加上一个触摸芯片，实现软开关，触摸开关
##32颗5mm白发粉雾灯，围成一个完整的心形，
###使用TYPE-C数据线供电，很方便的，插上数据线后是不亮的，需要用手摸一下正面的“always you” ,即可开始，在摸一下灯熄灭
####边框是CAD画的，很对称，光滑，每颗灯也是我一点一点测量距离，移动的，上下，左右间距对称
#####程序中有多种样式，起码两分钟花式
